So far, I have learned about Text Editors like Visual Studio Code. Text editors are essentially programs that allow us to write code. HTML (hypertext markup language) is one language that allows us to see text on a site. It basically consists of the title, head, and body with closing tags.

One key element to development is learning how to problem solve. Developers cannot memorize everything they learn, so there are plenty of shortcuts and Google can be a powerful tool. However, it is still important that they understand the foundation of what they do. For example, developers should know that a !Doctype HTML tag is a declaration that tells the browser what type of code language is being used, so that the server can portray this text or visual to the user. It is also important for developers to understand the foundation of coding languages, because language is not fixed. HTML has now evolved to HTML5.

Some additional things to know: there are approximately 10-15 HTML tags that are used which include: h tags (header), p tags(paragraph), ol tags (ordered list), li tags (list items), etc... A block of content with the tag are considered elements.

An ordered list tag or ol tag allows you to create a numerical text list. For example, a list of fruits to buy:

  1. apples
  2. oranges
  3. pears
An unordered list tag or ul tag allows you to create bullet points without the numerical text list. For examples, a list of why you love fruit:
  • healthy
  • tasty
  • full of fiber and vitamins
Regardless of the type of tags you utilize here, we are essentially nesting child tags within a parent one (e.g. ul tag (child) is nested with the p tag(parent)). You can even nest lists within lists. For example, a list of types of apples within your list of fruits to buy:
  1. apples
    • fuji
    • granny smith
    • macintosh
  2. oranges
  3. pears

Not all tags require a closing tag. There are some self-closing tags like ones used for page breaks or images. Some self closing tags also have attributes which creates a special property to that tag. An attribute always has a value attached to it. If we used the fruit list above and wanted to add an image here. We would utilize the self-closing tag for images to do so.

An image tag typically includes 2 attributes (alt and src). The alt attribute is an alternate text for the image for screen readers. The src attribute is the source of the image.
